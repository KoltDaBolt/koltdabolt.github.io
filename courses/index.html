Computer Networking

This course is designed to provide a full overview of computer networking, covering everything from the fundamentals of modern networking technologies and protocols to an overview of the cloud, as well as practical applications and network troubleshooting. Topics studied includes the layers of the TCP/IP and OSI network models (Physical, Data Link, Network, Transport, Session, Presesntation, Application), networking devices, IPv4 and IPv6 as well as subnetting, DNS and name resolution, DHCP, NAT, VPNs and Proxies and staying secure, the history of networking, wireless networking, the cloud, and basic network troubleshooting technologies.

Trees and Graphs

This course covers algorithms on tree data structures, binary search trees, graph structures, and basic traversal algorithms on graphs.

Web Dev II

Development of web pages and full stack applications using your JavaScript/TypeScript framework of choice. My tech stack of choice was the MEAN stack: MongoDB, ExpressJS, Angular, NodeJS. I also looked into other styling languages and frameworks such as SCSS, BootStrap, Bulma, etc. Other topics studied include web services and (REST) APIs, authentication vs. authorization, security, and scalability.

Python to Interact with the OS

This course, more focused on IT and automation senarios, covers using python to interact with the operating system, such as reading and writing files, file and directory management, dealing with CSV files, regular expressions, data streams, subprocesses, log files, and also covers automatic testing and how to handle errors and exceptions.

Python Fundamentals

Designed to teach you the foundations in order to write simple programs in python. Topics studied includes python syntax, and learning the built in data structures, such as a list, tuple, set, and dictionary.

Web Dev I

Development of web pages and full stack applications using the basic web technologies: HTML, CSS, JavaScript, jQuery, AJAX, PHP, and SQL.

Data Structures and Algorithms

The focus of this course was on the design of algorithms to solve nontrivial problems both correctly and efficiently. Topics studied includes assessing alternative solutions to given algorithmic problems, memory management, recursion, abstract data types, linked lists, stacks, queues, trees, and Big O computational complexity, as well as sorting and searching algorithms.

C Programming

Experience in developing algorithms, designing, and implementing programs. Topics include syntax/semantics, flow control, loops, recursion, I/O, arrays, strings, structs, command line arguments, and pointers.
